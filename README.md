# MP-BusinessCards-Pro

מערכת לניהול כרטיסי ביקור דיגיטליים — Backend ב-Node.js, Express, MongoDB, JWT.

## התחלה מהירה

```bash
cd src
npm install
```

העתק את `/.env.example` ל-`.env.development` (או `.env.production`) והגדר משתני סביבה. אין להעלות ל-Git קבצי `.env` עם סיסמאות אמיתיות.

```bash
npm start
```

- **אתר:** `http://localhost:8181` (ברירת מחדל בפיתוח)
- **API:** `http://localhost:8181/api`

תיעוד מלא (טכנולוגיות, משתמשי בדיקה, מבנה): [`src/README.md`](src/README.md).
